### ⚠️ 2026-02-26 15:21 追加: **根本原因の特定と修正**

| 試行 | 内容 | 結果 |
| :--- | :--- | :--- |
| 1回目 | `handleAuthLogin` の `uiInitialized` 引数修正 + `auth:login` リスナー移動 | 効果なし |
| 2回目 | `syncAllBooksFromCloud` の `didApplyIndex` 判定改善 + 未リンク書籍対応 | 効果なし |
| **3回目** | **`let index = {};` の宣言漏れを修正** | **テスト待ち** |

**根本原因**: 2回目の修正で `let index = {};` の行が誤って削除されたため、`syncAllBooksFromCloud` 関数内の `index` 変数が `ReferenceError` を発生させ、**同期処理全体が即座にクラッシュ**していた。これにより：
- クラウドからのインデックス取得（プル）が途中で中断
- ローカル書籍のクラウドへのアップロード（プッシュ）にも到達しない
- 結果として両端末のライブラリが全く統合されない

**修正内容**: `sync-logic.js` L223 に `let index = {};` を復元。
